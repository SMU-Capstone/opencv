# 가로쓰레기통 알리미 (OpenCV)
- 가로쓰레기통 알리미 앱의 OpenCV 파트 레포입니다.
- 본 프로젝트는 YOLO 모델을 base model로 사용합니다.
- 라즈베리파이 환경에서 Object detection과 image classification을 합니다.
- 플라스틱, 종이, 병, 캔, 고철, 스티로폼, 비닐 등 15가지 재활용 쓰레기 정보를 예측합니다.
